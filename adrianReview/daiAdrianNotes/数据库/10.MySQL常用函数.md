# COUNT函数说明（重点）

## 1.count(id)和count(字段)

- 如果这个字段是定义为 not null 的话，一行行地从记录里面读出这个字段，判断不能为null，按行累加
  - count(id) 一般就是 not null 的字段

- 如果这个字段定义允许为 null，那么执行的时候，判断到有可能是 null，还要把值取出来再判断一下，不是 null 才累加



## 2.count(1)和count(*)

​		count(1)和count(*)都会对全表进行扫描，统计所有记录的条数，包括那些为 null 的记录

​		但是数据库会对 count(*) 进行优化，自动选择最优的字段



## 区别和执行效率（重要）

一般来说：`count(*) >= count(1) > count(字段)`

1. 在<font color=red>**表没有主键时**</font>，count(1) 比 count(*) 快
2. <font color=red>**有主键时，并且主键作为计算条件**</font>，count(主键) 效率最高。即id作为表主键时，count(id) 效率最高
3. <font color=red>若表**只有一个字段**</font>，则 count(*) 效率较高
