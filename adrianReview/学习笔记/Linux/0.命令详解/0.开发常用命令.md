### 杀死进程

```shell
#首先可以通过ps查询到进程PID
ps -ef | grep java

#根据PID杀掉进程
kill -9 PID

```



### curl发送HTTP请求

```shell
# curl 命令是用来发送HTTP请求的
#参数：
-X  指定请求方式
-v  显示响应结果
-u  携带用户名/密码
-H  携带请求头信息

curl -X get -v -u 用户名:密码 -H "请求头信息" http:/www.dai.com/user/1
```
