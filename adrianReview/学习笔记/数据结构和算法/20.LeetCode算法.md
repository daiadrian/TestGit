### 快乐数

题目：一个“快乐数”定义为：对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和，然后重复这个过程直到这个数变为 1，也可能是无限循环但始终变不到 1。如果可以变为 1，那么这个数就是快乐数

> 例如：输入: 19 输出: true 
>
> 解释:
>
> 1^2 + 9^2 = 82
>
> 8^2 + 2^2 = 68
>
> 6^2 + 8^2 = 100
>
> 1^2 + 0^2 + 0^2 = 1

**<font color=red>思路：</font>** **当出现了跟输入的数字或者在重复过程过产生的数字一致时，那么就有出现了无限循环的情况了**

```java
public static boolean isHappyNumber(int n) {
    // Set 装不重复的数据
    Set<Integer> set = new HashSet<>();
    //n等于1的时候就是快乐数
    //set如果出现了包含之前出现过的数字, 那么就出现无限循环的情况
    while(n != 1 && !set.contains(n)){
        set.add(n);
        int sum = 0;
        //这个while循环是计算每次得到的值
        while(n > 0){
            // 整数取余 10 就等到个位数字
            sum += (n % 10) * (n % 10);
            n /= 10;
        }
        n = sum;
    }
    return  n == 1;
}
```
